# 던전 오피스 개인정보처리방침

- 대상 앱: 던전 오피스 Android 앱
- 지원 이메일: `2020promoking@gmail.com`

## 1. 앱이 수집하는 정보

던전 오피스 앱과 개발자는 앱 사용 과정에서 개인정보나 사용자 데이터를 수집하지 않습니다. 앱에는 계정, 로그인, 광고, 분석, 원격 크래시 수집, 푸시 알림, 원격 설정, 클라우드 동기화와 앱 내부 결제가 없습니다.

앱은 이름, 이메일 주소, 전화번호, 연락처, 위치, 광고 ID, 기기 식별자, 플레이 행동과 진단 정보를 개발자에게 전송하지 않습니다. 앱에는 게임 기능을 위한 인터넷 연결이 없으며, 원격 서버와 통신하지 않습니다.

## 2. 기기 안에 저장하는 정보

게임 진행, 최고 기록, 게임 내 골드, 설정과 진행 중 라운드는 사용자의 Android 기기에서 앱 전용 로컬 저장소에만 보관됩니다. 개발자는 이 데이터에 접근하거나 원격으로 복원할 수 없습니다.

사용자는 Android 설정에서 앱 저장공간을 삭제하거나 앱을 제거해 로컬 데이터를 삭제할 수 있습니다. 삭제한 진행을 복원하는 던전 오피스 계정이나 자체 클라우드 백업은 제공하지 않습니다.

## 3. Android·기기 제조사 백업

현재 배포 앱은 Android manifest의 `allowBackup=false` 설정으로 앱 데이터의 자동 백업을 허용하지 않도록 요청합니다. 별도의 클라우드 저장이나 동기화 기능도 제공하지 않습니다. 다만 Android 12 이상에서의 기기 간 전송과 Android 또는 기기 제조사가 제공하는 전체 기기 백업·복원·복제 도구는 앱 밖의 운영체제 기능이며 제조사별로 동작이 다를 수 있습니다. 따라서 제조사 구현과 사용자의 기기·계정 설정에 따라 로컬 앱 데이터가 처리될 가능성을 개발자가 완전히 통제한다고 보장할 수 없습니다.

이러한 플랫폼 처리는 던전 오피스 앱이나 개발자 서버로 데이터를 보내는 것이 아닙니다. 사용하는 Android, 기기 제조사 또는 백업 서비스의 개인정보처리방침과 설정을 확인해 주세요.

## 4. 제3자 제공과 공유

던전 오피스 앱은 사용자 데이터를 수집하지 않으므로 광고사, 분석사나 그 밖의 제3자에게 사용자 데이터를 판매하거나 공유하지 않습니다.

앱은 오픈소스 소프트웨어를 사용하지만, 포함된 구성요소가 사용자 데이터를 외부로 전송하도록 구성하지 않습니다. 사용 중인 오픈소스와 라이선스는 앱 내부 및 [오픈소스 라이선스 페이지](https://mrkimkim.github.io/dungeon-office/licenses/)에서 확인할 수 있습니다.

## 5. 지원 이메일

앱은 지원 이메일 주소를 표시하고 복사할 수 있게 하며, 사용자가 선택하면 OS의 메일 작성 화면을 열 수 있습니다. 앱이 메일을 자동으로 보내거나 진단 파일을 자동 첨부하지 않습니다.

사용자가 `2020promoking@gmail.com`으로 직접 메일을 보내면 이메일 주소, 본문과 사용자가 선택한 첨부물은 사용자의 메일 서비스와 Gmail을 통해 개발자에게 전달됩니다. 이는 앱의 자동 데이터 수집이 아닌 사용자의 자발적인 연락입니다.

개발자는 지원 메일을 문의 확인과 회신에만 사용하며 판매하거나 광고 목적으로 공유하지 않습니다. 지원 또는 법적 보관 필요가 끝나기 전까지 메일 계정에 보관될 수 있습니다. 지원 메일의 삭제를 요청하려면 같은 주소로 요청해 주세요. 메일 전송·보관에는 사용자와 Google의 메일 서비스 정책이 적용됩니다.

지원에 필요하지 않은 비밀번호, 결제 정보, 신분증과 같은 민감한 정보를 메일로 보내지 마세요.
