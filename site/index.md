# 던전 오피스 지원

던전 오피스는 Android용 오프라인 싱글 플레이 게임입니다. 현재 MVP의 R1~R5는 무료이며, 광고·계정·앱 내부 결제가 없습니다.

## 문의

- 지원 이메일: `2020promoking@gmail.com`
- [개인정보처리방침]({{ '/privacy/' | relative_url }})
- [오픈소스 라이선스]({{ '/licenses/' | relative_url }})

문의 메일에는 문제 상황, 앱 버전과 재현 절차만 적어 주세요. 비밀번호, 결제 정보, 신분증과 같이 지원에 필요하지 않은 개인정보는 보내지 마세요.

앱은 문의를 자동 전송하지 않습니다. 이메일 링크를 선택하면 사용자의 메일 앱으로 이동하며, 실제 전송과 보관에는 사용자의 메일 서비스 정책이 적용됩니다.

## 오프라인 이용

게임 실행, R1~R5 플레이, 저장·복구와 앱 내부 개인정보·오픈소스 고지 열람에는 네트워크가 필요하지 않습니다. 이 웹사이트는 공개 고지를 제공하기 위한 앱 외부 지원 채널이며, 사이트 접속 여부가 게임 이용에 영향을 주지 않습니다.

이 사이트에는 프로젝트가 추가한 광고, 분석 스크립트나 문의 폼이 없습니다. 사이트는 GitHub Pages가 호스팅하므로 웹 접속 정보는 [GitHub 개인정보처리방침](https://docs.github.com/site-policy/privacy-policies/github-general-privacy-statement)에 따라 처리될 수 있습니다.
